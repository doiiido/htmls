Análise de Indícios de Direcionamento para Fabricante

Esta análise identifica trechos do edital que podem indicar direcionamento para fabricante específico, com referência ao arquivo, página e linha.

Item Indício Possível Fabricante Fonte (Arquivo / Página / Linha)
Item 1 – Camada de Segurança (NGFW) Exigência de que todos os modelos de firewalls sejam do mesmo fabricante e compatíveis com itens de gerência. Indica solução integrada (fabricantes como Fortinet, Palo Alto, Check Point) PE18025_COM AVISO.pdf / p.20 / linha 3.1.12
Item 1 – Camada de Segurança (NGFW) Integração nativa com Let’s Encrypt para certificados SSL. Fortinet (FortiGate) é um dos poucos com suporte nativo a Let’s Encrypt. PE18025_COM AVISO.pdf / p.24 / linha 3.2.43
Item 1 – Camada de Segurança (NGFW) Conectores nativos para VMware ESXi, Cisco ACI e Kubernetes. Fortinet e Palo Alto possuem integração nativa com esses ambientes. PE18025_COM AVISO.pdf / p.24 / linha 3.2.45
Item 1 – Camada de Segurança (NGFW) Tokens móveis (5.000 por NGFW) sem custo adicional. Fortinet (FortiToken) é conhecido por oferecer tokens móveis integrados. PE18025_COM AVISO.pdf / p.24 / linha 3.2.49
Item 2 – Solução de Relatórios Appliance virtual deve ser do mesmo fabricante da solução NGFW. Indica fabricante que oferece NGFW + gerência integrada (Fortinet, Palo Alto, Check Point). PE18025_COM AVISO.pdf / p.31 / linha 4.1.1 e 4.1.5
Item 3 – Serviços Equipe certificada pelo fabricante da solução ofertada. Indica necessidade de certificação específica (ex.: NSE para Fortinet). PE18025_COM AVISO.pdf / p.37 / linha 7.2
Requisitos Gerais Obrigatoriedade de homologação ANATEL para appliances NGFW. Fabricantes com homologação ANATEL (Fortinet, Cisco, Palo Alto). PE18025_COM AVISO.pdf / p.24 / linha 3.1.24

Resumo da Análise

  • Há forte indício de direcionamento para fabricantes que oferecem solução integrada NGFW + gerência + tokens móveis, como Fortinet, Palo Alto ou Check Point.
  • Exigências como integração nativa com Let’s Encrypt, conectores para VMware/Cisco/Kubernetes e fornecimento de tokens móveis são características específicas de poucos fabricantes.
  • A obrigatoriedade de que todos os itens sejam do mesmo fabricante reforça a restrição de mercado.

Fonte: PE18025_COM AVISO.pdf, páginas 19–39 (Anexo I – Termo de Referência).
